# Plan 08-03: Charts and CSV Export

## Goal
Add performance trend charts and CSV export functionality.

## Requirements Covered
- REPT-01: Trend charts show performance over date range
- REPT-04: User can export reports as CSV

## Tasks

### 1. Create Simple Chart Component
- Use SVG for simple line/bar charts
- Or integrate lightweight chart library (recharts)
- Display metrics over time

### 2. Add Charts to Dashboard
- Spend over time
- Sales over time
- ACoS trend

### 3. Create CSV Export Functions (lib/export.ts)
- exportCampaignsCSV()
- exportKeywordsCSV()
- exportAuditLogCSV()

### 4. Add Export Buttons to Pages
- Campaigns page: Export to CSV
- Keywords page: Export to CSV
- Audit log: Export to CSV

### 5. Create Export API Endpoints
- GET /api/export/campaigns
- GET /api/export/keywords
- GET /api/export/audit

## Chart Requirements
- 7/30/90 day views
- Compare to previous period
- Responsive sizing

## CSV Format
Include all visible columns plus ID

## Verification
- [ ] Charts render on dashboard
- [ ] CSV exports download correctly
- [ ] Export includes all data

## Estimated Complexity
Medium
