# Plan 04-03: Campaign Table

## Goal
Create sortable, filterable campaign table with full metrics.

## Requirements Covered
- CAMP-01: View all campaigns (SP/SB/SD) in sortable, filterable table
- CAMP-02: Campaign table shows full metrics
- CAMP-03: Filter campaigns by type, status, performance range, date range

## Tasks

### 1. Install Table Component
Add shadcn/ui table or build custom.

### 2. Create Campaigns Page
New page at /campaigns with table.

### 3. Create Campaign Data Query
Fetch campaigns with aggregated metrics.

### 4. Implement Sorting
Client-side sorting by column.

### 5. Implement Filtering
Type filter (SP/SB/SD), Status filter (enabled/paused).

### 6. Add to Navigation
Link to Campaigns page in header.

| Column | Type | Sortable |
