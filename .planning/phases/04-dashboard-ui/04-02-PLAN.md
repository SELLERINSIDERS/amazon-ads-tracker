# Plan 04-02: Performance Cards & Alerts

## Goal
Create performance cards showing key metrics with trends.

## Requirements Covered
- DASH-01: Performance cards (spend, ACoS, ROAS, impressions, clicks, CTR, CPC, orders, sales)
- DASH-02: Cards show trends with % change
- DASH-05: Alerts section shows warnings
- DASH-06: Top/worst performers

## Tasks

### 1. Create MetricCard Component
Card showing metric value with trend indicator.

### 2. Create Dashboard Metrics Server Function
Aggregate metrics from CampaignMetric table.

### 3. Update Dashboard Page
Replace placeholder cards with real MetricCards.

### 4. Create Alerts Section
Show paused campaigns, high ACoS, budget warnings.

## Metrics Calculations

```typescript
// ACoS = (Spend / Sales) * 100
// ROAS = Sales / Spend
// CTR = (Clicks / Impressions) * 100
// CPC = Spend / Clicks
```
