# Plan 03-04: UI & Trigger

## Goal
Add manual sync trigger and status display to dashboard.

## Requirements Covered
- SYNC-04: User can trigger manual sync from dashboard
- SYNC-05: Dashboard shows last sync timestamp and status

## Tasks

### 1. Create Sync API Endpoint
POST /api/sync - Triggers sync for current profile.

### 2. Create Sync Status API
GET /api/sync/status - Returns current sync state.

### 3. Add Sync Button to Dashboard
Button to trigger manual sync with loading state.

### 4. Add Sync Status Display
Show last sync time, current status, any errors.

### 5. Update Dashboard Connection Card
Show sync info in the connection status area.

## UI Components

```tsx
// Sync status badge
<SyncStatusBadge status="idle|syncing|completed|failed" />

// Sync trigger button
<Button onClick={triggerSync} disabled={isSyncing}>
  {isSyncing ? "Syncing..." : "Sync Now"}
</Button>

// Last sync display
<p>Last synced: {lastSyncAt ? formatDate(lastSyncAt) : "Never"}</p>
```
