# Plan 05-03: Inline Editing

## Goal
Enable inline editing of campaigns and keywords.

## Requirements Covered
- CAMP-04: Inline edit campaign budget, bid, status

## Tasks

### 1. Create Edit Button Column
- Add "Edit" button to campaign and keyword tables
- Opens inline edit mode

### 2. Create Edit Modal/Inline Form
- Budget input (currency)
- Status toggle (enabled/paused)
- Save/Cancel buttons

### 3. Create Update API Endpoints
- `PATCH /api/campaigns/[id]`
- `PATCH /api/keywords/[id]`
- Store changes locally
- Attempt Amazon API update (may fail without credentials)

### 4. Add Optimistic Updates
- Update UI immediately
- Show pending state
- Handle errors

## Edit Fields

**Campaign:**
- Budget (daily)
- Status (enabled/paused)

**Keyword:**
- Bid
- Status (enabled/paused)
